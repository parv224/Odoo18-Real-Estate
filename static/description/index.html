Real Estate

Manage properties and offers in Odoo

This module provides a simple solution to manage real estate properties and property offers inside Odoo. It is designed to keep property data, offers, and sale states organized in a clean and structured way.

Main Features

  • Create and manage real estate properties
  • Define property types and tags
  • Manage property offers
  • Accept or reject offers
  • Automatic property state updates

Usage

Users can create properties, receive offers from buyers, and manage the full property lifecycle from availability to sale.
